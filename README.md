## About The Project

Swifty-Protein is an iOS application to visualize Ligands listed by the [Protein Data Bank (PDB)](https://www.rcsb.org/) with a Ball and Sticks model. Application allows users to zoom, rotate, and share visualizations. Utilizes TouchID to provide a secure authentication process.



<!-- GETTING STARTED -->
## Getting Started
No outside dependencies are required to build or run the application, outside of XCode.

### Prerequisites

An up-to-date XCode installation is required to run the application simulation

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Application can be run inside XCode or loaded onto an iOS device.
